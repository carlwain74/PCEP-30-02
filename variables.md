# Variables

## Summary

* Store data values
* No type required
* Requires an initial value even if empty
* Can assign a value to multiple variables
* Used `del` to delete a variable

```
	a = 100
	b = "Hello"
	c = d = 200

	a,b = 100,"Hello"

	del a
```
